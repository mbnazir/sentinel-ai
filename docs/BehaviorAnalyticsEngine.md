# Behavior Analytics Engine

Milestone 7 introduces longitudinal behavior analytics.

## Purpose

Session risk answers:

> Is this session suspicious?

Behavior analytics answers:

> Is this person, supervisor, manager, campaign, or site developing a suspicious pattern?

## Initial profiles

- Agent behavior profile
- Supervisor behavior profile

Future profiles:

- Manager
- Payroll user
- Campaign
- Site
- Client
- Location

## Metrics

Initial metrics include:

- session count
- average risk score
- max risk score
- high-risk session count
- inserted activities
- deleted activities
- extended activities
- payroll adjustments
- manual added minutes
- rule count

## Trend detection

Milestone 7 includes deterministic slope-based trend detection.

## Peer comparison

Milestone 7 includes z-score based peer deviation.

## Important HR principle

Behavior analytics is not an accusation engine. It identifies patterns requiring review.
